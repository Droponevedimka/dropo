📨 Удалите прокси Telegram

Приложение закрывается и локальный прокси перестанет работать. Чтобы Telegram продолжил подключаться, удалите сохранённый прокси dropo (127.0.0.1:1443) — программно убрать его нельзя.

Как удалить:
1. Telegram → Настройки → Продвинутые настройки
2. Раздел «Данные и память» → Тип соединения
3. Настройки прокси → у прокси MTPROTO 127.0.0.1:1443 нажмите ⋮ → Удалить
нажмите на скрин, чтобы увеличить

Какой трафик направлять через VPN
Через VPN идут только заблокированные сайты (РКН + сервисы, блокирующие РФ). Минимальная нагрузка на VPN.
Открывать все иностранные сайты через VPN/обход
По умолчанию выключено. Когда включено, RU-сервисы остаются напрямую, остальное идет через ByeDPI или VLESS с меньшей задержкой.

Как Dropo подключается к сетевому слою Windows
Auto использует Deep Windows через winws/WinDivert, если компоненты доступны, иначе включает Compatibility TUN.
Адрес прокси для RU-трафика
Необязательно. Без адреса RU-трафик пойдёт через основной VPN, если он настроен. Не используйте адреса из публичных бесплатных списков прокси.

Не использовать бесплатные методы

Тест доступности сервисов
