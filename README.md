# dropo

**dropo** — клиент для обхода блокировок и подключения к VPN. На Windows приложение
сначала использует бесплатные методы обхода, а при наличии VPN-подписки может
переключаться на неё там, где бесплатный способ не помогает.

> [!WARNING]
> Единственный официальный источник dropo — репозиторий
> [Droponevedimka/dropo](https://github.com/Droponevedimka/dropo). Не устанавливайте
> сборки из сторонних источников.

## Быстрый старт на Windows

1. Скачайте ZIP-архив и полностью распакуйте его в отдельную папку.
2. Запустите `dropo.exe` и подтвердите запрос Windows на запуск от имени
   администратора.
3. При первом запуске дождитесь загрузки необходимых компонентов.
4. Нажмите кнопку подключения. VPN-подписку можно добавить позже в приложении —
   для бесплатных методов обхода она не обязательна.

Windows может предложить установить сертификат издателя dropo для текущего
пользователя. Это убирает предупреждения о неизвестном издателе; установка
выполняется только после вашего подтверждения.

## Возможности

- бесплатный обход блокировок на Windows;
- VPN-подписки VLESS, VMess, Trojan, Shadowsocks, Hysteria2 и TUIC;
- WireGuard;
- отдельные правила подключения для разных сервисов;
- поддержка Telegram через локальный прокси на Windows и VPN-маршрут на Android;
- профили, статистика, логи и проверка обновлений.

## Как маршрутизируется трафик

> [!IMPORTANT]
> Начиная с версии **2.1.55** Windows-сборка использует только
> [bol-van/zapret2](https://github.com/bol-van/zapret2) и `winws2.exe`.
> Старый `bol-van/zapret` (zapret1), `winws.exe` и интерфейс
> `--dpi-desync` больше не входят в dependencies и не запускаются приложением.

- На Windows используется один режим **Windows Unified**: sing-box TUN отвечает
  за маршрутизацию, а один процесс zapret2 `winws2.exe` содержит отдельный
  профиль для каждого поддерживаемого заблокированного сервиса.
- Для каждого сервиса стратегии проверяются в заданном порядке до первого
  успешного результата. Выбор сохраняется отдельно от других сервисов; сбой
  одного сервиса не заменяет стратегию остальных.
- Если ни одна стратегия сервиса не сработала, используется VPN-подписка. Такой
  отрицательный результат временный: проверка повторяется через 30 минут или
  сразу после смены сети.
- На Android бесплатного обхода пока нет: сервисы, которым нужен обход,
  направляются через добавленную VPN-подписку.
- Российские сервисы по умолчанию идут напрямую, без VPN.
- Приложения внутри Dropo Space работают отдельно от VPN основного профиля и
  предназначены только для прямого доступа.

> [!NOTE]
> Версия для Android пока является MVP и требует добавленной VPN-подписки.
> Бесплатные методы обхода без подписки на Android сейчас не используются.

## Dropo Space на Android

Dropo Space создаёт отдельный рабочий профиль для приложений, которым может
мешать включённый системный VPN. Приложения и их данные внутри этого профиля
отделены от обычных копий и не используют VPN основного профиля. Добавляйте туда
только приложения, которым нужен обычный прямой доступ.

1. Сначала установите нужные приложения в основном профиле телефона.
2. В dropo откройте **Ещё → Dropo Space** и нажмите
   **Создать Dropo Space**.
3. Подтвердите создание рабочего профиля в системном мастере Android и дождитесь
   его завершения. dropo автоматически добавит поддерживаемые приложения,
   которые уже установлены на телефоне.
4. Вернитесь в dropo и нажмите **Обновить**. Готовые приложения будут отмечены
   статусом **в Dropo Space**.
5. Если какое-то приложение не добавилось, нажмите **Добавить** напротив него.
   dropo попробует восстановить рабочий профиль автоматически. Если прошивка
   запретит установку, появится справочное окно: переход к ручной установке
   выполняется только после вашего подтверждения.
6. При предложении Android разрешите создать ярлык. Запускайте рабочую копию по
   ярлыку dropo или со вкладки **Рабочие** в списке приложений телефона.

Вход в аккаунт и данные обычной копии не переносятся — в Dropo Space нужно
авторизоваться заново. На Xiaomi рабочие приложения обычно помечены значком
портфеля. Если Dropo Space приостановлен, включите рабочий профиль кнопкой с
портфелем в системной панели и нажмите **Обновить** в dropo.

dropo создан с использованием открытых проектов и компонентов:

| [Xray-core](https://github.com/XTLS/Xray-core) | Поддержка отдельных VPN-транспортов |
| TG WS Proxy | Локальный прокси для Telegram |
| [Flutter](https://github.com/flutter/flutter) | Пользовательский интерфейс |

Если вы хотите поддержать эти проекты, используйте только способы, указанные
самими авторами на их официальных GitHub-страницах — например, кнопку
**Sponsor**, если она доступна. Не переводите средства людям, которые выдают себя
за авторов сторонних компонентов.

## Если возникла проблема

- Не запускайте `dropo.exe` прямо из ZIP-архива — сначала распакуйте все файлы.
- Отключите другие VPN-приложения: одновременная работа может вызвать конфликт.
- Если не загрузились компоненты, проверьте доступ к GitHub и повторите попытку.
- Логи Windows находятся в `%LOCALAPPDATA%\dropo\logs\`.
